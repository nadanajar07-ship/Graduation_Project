FAQ

Frequently asked questions about the system

Getting Started
How do I log in for the first time?
Use credentials provided by your Admin. On first login you'll be prompted to change your password and enable two-step verification (JWT + TOTP).
What are the different user roles?
Admin (full access), Manager (team & project), and Employee (personal workspace). Access is enforced via RBAC with JIT tokens.
How does the time tracking work?
Sessions are tracked automatically when you're active. You can also start/stop a timer manually on any task from the task detail page.
AI & Analytics
What is AI Performance Clustering?
The ML model uses K-Means clustering to group employees (High / Average / Needs Attention) based on activity vectors: work hours, task completion, idle time, and keystrokes.
How are anomalies detected?
The system uses unsupervised anomaly detection to flag deviations from normal patterns — like unusual idle spikes or sudden productivity drops.
Privacy & Security
Are screenshots visible to other employees?
No. Screenshots are only accessible to Admins and direct Managers. Regular employees cannot view monitoring data of their peers.
Is my data encrypted?
Yes. All data is encrypted in transit (HTTPS/TLS) and at rest (AES-256). Auth uses JWT with short-lived tokens. Passwords hashed with bcrypt. System is GDPR & ISO 27001 compliant.
Tasks & Projects
Can the AI assign tasks automatically?
Yes. The AI Task Suggestion engine analyzes current workloads and skill matches to recommend the best assignee. Managers can accept or override suggestions.
How do I create a workflow automation rule?
Go to Settings → Workflow Automation. Use the no-code rule builder: define a trigger (e.g. task status changes) and an action (e.g. send notification, reassign, move column).
What file types can I attach to tasks?
Images, PDFs, documents, and code files up to 25 MB each. All attachments are stored in AWS S3 with access control policies.
